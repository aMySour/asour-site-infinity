How does this work?

Infinity combines PIXI.js, THREE.js and the HTML DOM. We use THREE.js for the background, PIXI.js for the world and for navigation, and real HTML elements for the UI.

The code of a page looks something like this:

            <head>
                <title>Cool Page</title>
            </head>
            <world>
                <p xy="0 0">Hello world!</p>
            </world>

As you can probably gather, Infinity aims to make it as easy as possible to make a page.
